MI PRACTICA DE GIT

que es GIT

Git es un sistema de control de versiones distribuido (DVCS) gratuito y de código abierto que permite a los desarrolladores rastrear los cambios en el código fuente de un proyecto. Es fundamental para la colaboración en equipo, ya que permite a varios desarrolladores trabajar simultáneamente en el mismo proyecto sin conflictos, creando y fusionando ramas de código de manera eficiente.
